# Twitter Clone

Features:

Sign up/Login
Posting Tweets
Deleting Tweets
Retweeting
Replying to Tweets
Tweets as threads
Following Profiles
Home Feed
User Profiles
Who to Follow

What I learned:
Firebase Database Structure
Firebase Transactions
Various querying methods
React Router Dom
React Memo
etc.

Used Context API and reducers to set username globally, but turned out redundant, because it doesn't persist on site refresh.
